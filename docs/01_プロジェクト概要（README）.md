# 🏷️ Enterprise-IT-Helpdesk-AI
🤖 社内IT・情シス向けBOT（問い合わせ／運用）

---

このプロジェクトは、企業内のITヘルプデスク業務を支援するためのAIボットシステムです。複数のAIエージェントが連携して、IT問い合わせの処理、運用支援、自動化を実現します。

## ⭐ 主な特徴

- 🖥️ **クロスプラットフォーム対応**: Ubuntu Linux と Windows 11 の両方で動作
- 💻 **CLI中心設計**: コマンドラインインターフェースを優先
- 👥 **役割分担**: 🤖 Claude、⚙️ GPT、🔍 Gemini/Perplexity などのAIエージェントが各々の役割を果たす
- 🛡️ **エンタープライズ向け**: 監査対応、セキュリティ重視の設計

## 📂 プロジェクト構造

- 📚 `docs/` - プロジェクトドキュメント
- ⚙️ `.opencode/` - OpenCode エージェント設定
- 📋 `AGENTS.md` - エージェント向けガイドライン

## 🚀 開発開始

📝 このプロジェクトは現在設計・ドキュメント段階です。実装フェーズでは以下の技術スタックを検討しています：

- 🎨 フロントエンド: React/TypeScript (必要に応じて)
- 🔧 バックエンド: Node.js/Python (AI連携用)
- ☁️ インフラ: Docker, Kubernetes (運用環境)
- 🔄 CI/CD: GitHub Actions

詳細な実装計画については `docs/` フォルダ内のドキュメントを参照してください。

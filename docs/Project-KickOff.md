## 💻 Ubuntu / Windows11 両対応のコツ

### 共通ルール（必須）

* CLI前提（GUI依存しない）
* パスは相対パス
* PowerShell + Python 併用
* `.env` で環境差分吸収
* OS依存処理は分離

### 実装例

* Ubuntu：

  * ClaudeCode / Gemini CLI 中心
* Windows11：

  * Codex CLI + PowerShell 中心

👉 **設計はLinux、実装はどちらでもOK**が理想
